## Description

Docs gardening mechanisms (from GSD): stamp generated kb docs with the commit SHA they describe so staleness is computable (git diff SHA..HEAD), plus a doc-verifier agent that checks doc claims against the live filesystem. Could run as a git hook or scheduled skill.
