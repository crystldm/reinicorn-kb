## Description

Task-level model tier requirements - exec plan tasks should specify whether they need flagship/reasoning models or can use fast/cheap models. The harness should define tiers (e.g. flagship, standard, fast) and let .reins-config or extensions map those to actual provider-specific models. Keeps the system provider-agnostic while letting teams optimize cost and latency per task.
